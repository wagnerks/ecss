<div class="textblock"><p>Doxygen supports most of the XML commands that are typically used in C# code comments. The XML tags are defined in Appendix D of the <a href="https://ecma-international.org/publications-and-standards/standards/ecma-334/">ECMA-334</a> standard, which defines the C# language. Unfortunately, the specification is not very precise and a number of the examples given are of poor quality.</p>
<p>Here is the list of tags supported by Doxygen:</p>
<table class="markdownTable">
<tr class="markdownTableHead">
<th class="markdownTableHeadLeft">XML Command</th><th class="markdownTableHeadLeft">Description</th></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_c"></a><a id="a649__3Cc_3E" name="a649__3Cc_3E"></a> <span class="tt"> &lt;c&gt;</span></td><td valign="top">Identifies inline text that should be rendered as a piece of code. Similar to using <span class="tt">&lt;tt&gt;</span>text<span class="tt">&lt;/tt&gt;</span>.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_code"></a><a id="a650__3Ccode_3E" name="a650__3Ccode_3E"></a> <span class="tt"> &lt;code&gt;</span></td><td valign="top">Set one or more lines of source code or program output. Note that this command behaves like <a class="el" href="commands.html#cmdcode">\code</a> ... <a class="el" href="commands.html#cmdendcode">\endcode</a> for C# code, but it behaves like the HTML equivalent <a class="el" href="htmlcmds.html#htmltag_CODE">&lt;CODE&gt;</a>...<a class="el" href="htmlcmds.html#htmltag_endCODE">&lt;/CODE&gt;</a> for other languages.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_description"></a><a id="a651__3Cdescription_3E" name="a651__3Cdescription_3E"></a> <span class="tt"> &lt;description&gt;</span></td><td valign="top">Part of a <a class="el" href="#xmltag_list">&lt;list&gt;</a> command, describes an item.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_example"></a><a id="a652__3Cexample_3E" name="a652__3Cexample_3E"></a> <span class="tt"> &lt;example&gt;</span></td><td valign="top">Marks a block of text as an example, ignored by Doxygen.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_exception"></a><a id="a653__3Cexception_3E" name="a653__3Cexception_3E"></a> <span class="tt"> &lt;exception cref="member"&gt;</span></td><td valign="top">Identifies the exception a method can throw.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_include"></a><a id="a654__3Cinclude_3E" name="a654__3Cinclude_3E"></a> <span class="tt"> &lt;include&gt;</span></td><td valign="top">Can be used to import a piece of XML from an external file. Ignored by Doxygen at the moment.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_inheritdoc"></a><a id="a655__3Cinheritdoc_3E" name="a655__3Cinheritdoc_3E"></a> <span class="tt"> &lt;inheritdoc&gt;</span></td><td valign="top">Can be used to insert the documentation of a member of a base class into the documentation of a member of a derived class that reimplements it.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_item"></a><a id="a656__3Citem_3E" name="a656__3Citem_3E"></a> <span class="tt"> &lt;item&gt;</span></td><td valign="top">List item. Can only be used inside a <a class="el" href="#xmltag_list">&lt;list&gt;</a> context.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_list"></a><a id="a657__3Clist_3E" name="a657__3Clist_3E"></a> <span class="tt"> &lt;list type="type"&gt;</span></td><td valign="top">Starts a list, supported types are <span class="tt">bullet</span> or <span class="tt">number</span> and <span class="tt">table</span>. A list consists of a number of <span class="tt">&lt;item&gt;</span> tags. A list of type table, is a two column table which can have a header.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_listheader"></a><a id="a658__3Clistheader_3E" name="a658__3Clistheader_3E"></a> <span class="tt"> &lt;listheader&gt;</span></td><td valign="top">Starts the header of a list of type "table".</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_para"></a><a id="a659__3Cpara_3E" name="a659__3Cpara_3E"></a> <span class="tt"> &lt;para&gt;</span></td><td valign="top">Identifies a paragraph of text.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_param"></a><a id="a660__3Cparam_3E" name="a660__3Cparam_3E"></a> <span class="tt"> &lt;param name="paramName"&gt;</span></td><td valign="top">Marks a piece of text as the documentation for parameter "paramName". Similar to using <a class="el" href="commands.html#cmdparam">\param</a>.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_paramref"></a><a id="a661__3Cparamref_3E" name="a661__3Cparamref_3E"></a> <span class="tt"> &lt;paramref name="paramName"&gt;</span></td><td valign="top">Refers to a parameter with name "paramName". Similar to using <a class="el" href="commands.html#cmda">\a</a>.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_permission"></a><a id="a662__3Cpermission_3E" name="a662__3Cpermission_3E"></a> <span class="tt"> &lt;permission&gt;</span></td><td valign="top">Identifies the security accessibility of a member. Ignored by Doxygen.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_remarks"></a><a id="a663__3Cremarks_3E" name="a663__3Cremarks_3E"></a> <span class="tt"> &lt;remarks&gt;</span></td><td valign="top">Identifies the detailed description. </td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_returns"></a><a id="a664__3Creturns_3E" name="a664__3Creturns_3E"></a> <span class="tt"> &lt;returns&gt;</span></td><td valign="top">Marks a piece of text as the return value of a function or method. Similar to using <a class="el" href="commands.html#cmdreturn">\return</a>.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_see"></a><a id="a665__3Csee_3E" name="a665__3Csee_3E"></a> <span class="tt"> &lt;see cref="member"&gt;</span></td><td valign="top">Refers to a member. Similar to <a class="el" href="commands.html#cmdref">\ref</a>.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_seealso"></a><a id="a666__3Cseealso_3E" name="a666__3Cseealso_3E"></a> <span class="tt"> &lt;seealso cref="member"&gt;</span></td><td valign="top">Starts a "See also" section referring to "member". Similar to using <a class="el" href="commands.html#cmdsa">\sa</a> member.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_summary"></a><a id="a667__3Csummary_3E" name="a667__3Csummary_3E"></a> <span class="tt"> &lt;summary&gt;</span></td><td valign="top">In case this tag is used outside a <a class="el" href="htmlcmds.html#htmltag_DETAILS">&lt;DETAILS&gt;</a> tag this tag identifies the brief description. Similar to using <a class="el" href="commands.html#cmdbrief">\brief</a>. In case this tag is used inside a <a class="el" href="htmlcmds.html#htmltag_DETAILS">&lt;DETAILS&gt;</a> tag this tag identifies the heading of the <a class="el" href="htmlcmds.html#htmltag_DETAILS">&lt;DETAILS&gt;</a> tag.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_term"></a><a id="a668__3Cterm_3E" name="a668__3Cterm_3E"></a> <span class="tt"> &lt;term&gt;</span></td><td valign="top">Part of a <a class="el" href="#xmltag_list">&lt;list&gt;</a> command.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_typeparam"></a><a id="a669__3Ctypeparam_3E" name="a669__3Ctypeparam_3E"></a> <span class="tt"> &lt;typeparam name="paramName"&gt;</span></td><td valign="top">Marks a piece of text as the documentation for type parameter "paramName". Similar to using <a class="el" href="commands.html#cmdparam">\param</a>.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_typeparamref"></a><a id="a670__3Ctypeparamref_3E" name="a670__3Ctypeparamref_3E"></a> <span class="tt">&lt;typeparamref name="paramName"&gt;</span></td><td valign="top">Refers to a parameter with name "paramName". Similar to using <a class="el" href="commands.html#cmda">\a</a>.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_value"></a><a id="a671__3Cvalue_3E" name="a671__3Cvalue_3E"></a> <span class="tt"> &lt;value&gt;</span></td><td valign="top">Identifies a property. Ignored by Doxygen.</td></tr>
<tr>
<td valign="top"><a class="anchor" id="xmltag_CDATA"></a><a id="a672__3C_21_5BCDATA_5B..._5D_5D_3E" name="a672__3C_21_5BCDATA_5B..._5D_5D_3E"></a> <span class="tt"> &lt;![CDATA[...]]&gt;</span></td><td valign="top">The text inside this tag (on the ...) is handled as normal Doxygen comment except for the XML special characters <span class="tt">&lt;</span>, <span class="tt">&gt;</span> and <span class="tt">&amp;</span> that are used as if they were escaped.</td></tr>
</table>
<p>Here is an example of a typical piece of code using some of the above commands:</p>
<div class="fragment"><div class="line"><span class="comment">/// &lt;summary&gt;</span><span class="comment"></span></div>
<div class="line"><span class="comment">/// A search engine.</span></div>
<div class="line"><span class="comment">/// &lt;/summary&gt;</span></div>
<div class="line"><span class="keyword">class </span>Engine</div>
<div class="line">{<span class="comment"></span></div>
<div class="line"><span class="comment">  /// &lt;summary&gt;</span></div>
<div class="line"><span class="comment">  /// The Search method takes a series of parameters to specify the search criterion</span></div>
<div class="line"><span class="comment">  /// and returns a dataset containing the result set.</span></div>
<div class="line"><span class="comment">  /// &lt;/summary&gt;</span></div>
<div class="line"><span class="comment">  /// &lt;param name=&quot;connectionString&quot;&gt;the connection string to connect to the</span></div>
<div class="line"><span class="comment">  /// database holding the content to search&lt;/param&gt;</span></div>
<div class="line"><span class="comment">  /// &lt;param name=&quot;maxRows&quot;&gt;The maximum number of rows to</span></div>
<div class="line"><span class="comment">  /// return in the result set&lt;/param&gt;</span></div>
<div class="line"><span class="comment">  /// &lt;param name=&quot;searchString&quot;&gt;The text that we are searching for&lt;/param&gt;</span></div>
<div class="line"><span class="comment">  /// &lt;returns&gt;A DataSet instance containing the matching rows. It contains a maximum</span></div>
<div class="line"><span class="comment">  /// number of rows specified by the maxRows parameter&lt;/returns&gt;</span></div>
<div class="line">  <span class="keyword">public</span> DataSet Search(<span class="keywordtype">string</span> connectionString, <span class="keywordtype">int</span> maxRows, <span class="keywordtype">int</span> searchString)</div>
<div class="line">  {</div>
<div class="line">    DataSet ds = <span class="keyword">new</span> DataSet();</div>
<div class="line">    <span class="keywordflow">return</span> ds;</div>
<div class="line">  }</div>
<div class="line">}</div>
